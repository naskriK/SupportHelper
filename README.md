## Getting Started

Follow the example

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/naskriK/SupportHelper.git
   ```
2. Install dependencies

```sh
  npm install
```

3. Run a server

```sh
  npm run dev
```
